# fuji-app

Demo app that shows Flickr images of Mount Fuji in a slideshow. Built by [Screwdriver](https://screwdriver.cd).

To setup:
```bash
$ git clone git@github.com:screwdriver-cd/fuji-app.git
$ cd fuji-app/app/
$ npm install
```

To start:
```bash
$ cd fuji-app/app/
$ npm start
```
Navigate to http://localhost:7000.


To test:
```bash
$ cd fuji-app/app/
$ npm test
```
